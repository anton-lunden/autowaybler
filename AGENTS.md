# AGENTS.md

## Development

- Run `npm install` to install dependencies.
- Run `npm run dev` to start with file watching.
- Run `npm run build` to compile TypeScript.
- Run `npm start` to run the compiled output.

## Commits

- Use [Conventional Commits](https://www.conventionalcommits.org/) (`feat:`, `fix:`, `chore:`, `docs:`).
- Keep subjects short and imperative: `fix: handle expired auth token`.
